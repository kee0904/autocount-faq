# AutoCount POS FAQ Hub

A searchable static FAQ landing page for AutoCount POS, FnB, Optical POS, QR Ordering, PalmPOS, OneSales, release notes, and help/reference categories.

## Local preview

```bash
npm install
npm run start
```

## Deploy

This project is configured for Vercel with Vite. Vercel should use:

- Build command: `npm run build`
- Output directory: `dist`
